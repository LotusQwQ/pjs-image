# PJS 表情模板

此为《绘名号高性能！》bot 中 pjs 功能使用的表情模板，包括官方表情和一些民间表情。

这些图片是 AI 抠完字后再手动细改的，所有图片与某人所谓的通过意想不到的方式获取原图无关。此仓库的目的就是让所有制作表情包插件的开发者都方便获取底图，平日时间不多随缘更新，谢谢理解。

## 使用

图片位于 `img/` 目录，按角色分文件夹存放。
